An online quiz system for Fundamental Mathematics

Project Overview

A preview individual assignment from Java2, Semester B/2025. Mathz is an interactive math quiz application developed to create an engaging learning environment.

It is integrated with a leaderboard system for peer-to-peer comparison and an admin panel for comprehensive content management.

Students (players) can attempt quizzes and track their performance, while administrators can manage questions and oversee leaderboard entries.

Key Features

  • User Authentication supporting both Player and Admin login roles.
  • Multiple-choice quiz functionality with automatic scoring.
  • Leaderboard System with automatic recording and real-time search and filtering of entries.
  • Admin Panel for quiz management (creating, updating, and deleting questions).
  • Leaderboard Management tools for deleting selected or all entries.

Technologies Used

The system is developed using JavaFX for the application interface and MySQL as the backend database. The design follows an Object-Oriented structure.
