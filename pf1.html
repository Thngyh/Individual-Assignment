Project Overview

A preview individual project from Introduction to Information Technology course, Semester A/2025.

A project management application designed to streamline task and project tracking for software engineering students and professionals.

It was created to provide a simplified, intuitive interface, as the complex interfaces of traditional tools can be overwhelming, making task and project management more approachable and less time-consuming.

Key Features

  • Task and Project Categorization.
  • Detailed task and project forms.
  • User-friendly interface with defined navigation and header components.
  • Solutions for software engineering teams.

Technologies Used

Developed using Microsoft Power Apps , which is a low-code platform. It leverages SharePoint lists to manage the project data, ensuring seamless integration and accessibility.
